# 1. Traversal Operation

## Q: What is Traversal?

=> Traversal means visiting every element of an array exactly once.

Example:

```java
int[] arr = {10, 20, 30, 40, 50};

for(int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

Output:

```text
10
20
30
40
50
```

---

## Time Complexity

```text
O(n)
```

Reason:

```text
Every element is visited once.
```

---

## Real Applications

```text
-> Displaying Student Records

-> Printing Reports

-> Processing Sensor Data

-> Reading Database Results

-> Machine Learning Dataset Processing
```

# 4. Searching Operation

## Q: What is Searching?

=> Finding whether a particular element exists in an array.

---

# Linear Search

## Definition

=> Check each element one by one until the target is found.

Example:

```java
int[] arr = {10,20,30,40,50};

int key = 30;

for(int i=0;i<arr.length;i++) {

    if(arr[i] == key) {

        System.out.println("Found at Index: " + i);
        break;
    }
}
```

Output:

```text
Found at Index: 2
```

---

## Complexity

```text
Best Case  : O(1)

Average    : O(n)

Worst Case : O(n)
```

---

# Binary Search

## Definition

=> Searches by repeatedly dividing the search space into halves.

Requirement:

```text
Array must be sorted.
```

Example:

```text
10 20 30 40 50 60 70
```

Search:

```text
50
```

Process:

```text
Middle = 40

50 > 40

Search Right Half

Found 50
```

---

## Complexity

```text
O(log n)
```

---

## Real Applications

```text
-> Database Searching

-> Search Engines

-> Dictionary Applications

-> Library Management Systems
```

# 5. Insertion Operation

## Q: What is Insertion?

=> Adding a new element into an array.

Example:

Before:

```text
10 20 40 50
```

Insert:

```text
30
```

After:

```text
10 20 30 40 50
```

---

## Algorithm

```text
Step 1: Find insertion position

Step 2: Shift elements right

Step 3: Insert new element
```

---

## Example

```java
int[] arr = {10,20,30,40,0};

for(int i = 3; i >= 2; i--) {
    arr[i+1] = arr[i];
}

arr[2] = 25;
```

Output:

```text
10 20 25 30 40
```

---

## Complexity

```text
O(n)
```

Reason:

```text
Shifting required.
```

---

## Real Applications

```text
-> Adding New Student Record

-> Adding Product to Inventory

-> Adding Transaction Entry
```

---

# 6. Deletion Operation

## Q: What is Deletion?

=> Removing an element from an array.

Example:

Before:

```text
10 20 30 40 50
```

Delete:

```text
30
```

After:

```text
10 20 40 50
```

---

## Algorithm

```text
Step 1: Find element

Step 2: Shift remaining elements left

Step 3: Reduce logical size
```

---

## Example

```java
int[] arr = {10,20,30,40,50};

for(int i = 2; i < arr.length - 1; i++) {

    arr[i] = arr[i + 1];
}
```

Output:

```text
10 20 40 50
```

---

## Complexity

```text
O(n)
```

---

## Real Applications

```text
-> Removing Student Record

-> Removing Product

-> Deleting Transaction History
```

# 8. Reverse Operation

## Q: What is Reversing?

=> Rearranging elements from opposite direction.

Before:

```text
10 20 30 40 50
```

After:

```text
50 40 30 20 10
```

---

## Algorithm

```text
Swap First ↔ Last

Move Inward

Repeat Until Center
```

---

## Example

```java
int start = 0;
int end = arr.length - 1;

while(start < end) {

    int temp = arr[start];

    arr[start] = arr[end];

    arr[end] = temp;

    start++;
    end--;
}
```

---

## Complexity

```text
O(n)
```

# 12. Finding Maximum Element

