# Conditional Statements in Java

---

# 1. Introduction to Conditional Statements

## What are Conditional Statements?

Conditional statements are decision-making structures in Java that allow a program to execute different blocks of code based on whether a condition is true or false.

They help programs make intelligent decisions instead of executing every statement sequentially.

Without conditional statements, programs would execute instructions one after another without any decision-making capability.

Example:

```java
int age = 20;

if(age >= 18) {
    System.out.println("Eligible to Vote");
}
```

Output:

```text
Eligible to Vote
```

---

## Why Do We Need Conditional Statements?

Conditional statements help us:

- Make decisions
- Execute specific code blocks
- Control program flow
- Implement business logic
- Build dynamic applications

Real-World Examples:

- ATM checking balance
- Login authentication
- Student grading system
- E-commerce discount calculation
- Banking transactions
- Access control systems

---

# How Conditional Statements Work

General Flow:

```text
Condition
    │
    ▼
 TRUE ? ----------► Execute Block
    │
    ▼
 FALSE
    │
    ▼
 Skip Block
```

Example:

```java
int marks = 75;

if(marks >= 40) {
    System.out.println("Pass");
}
```

Condition:

```java
marks >= 40
```

Result:

```text
true
```

Therefore block executes.

---

# Boolean Expressions

Java conditions must always evaluate to:

```java
true
```

or

```java
false
```

Example:

```java
int age = 20;

System.out.println(age >= 18);
```

Output:

```text
true
```

---

## Invalid Conditions

Java does NOT allow:

```java
if(1)
```

or

```java
if(0)
```

Unlike C or C++, Java requires a boolean result.

Wrong:

```java
if(1) {
}
```

Compiler Error.

Correct:

```java
if(true) {
}
```

# 2. Simple if Statement

Executes code only when condition is true.

Syntax:

```java
if(condition) {

    // code
}
```

Example:

```java
int age = 20;

if(age >= 18) {

    System.out.println("Adult");
}
```

Output:

```text
Adult
```

---

## Execution Flow

```text
Condition
    │
    ▼
 TRUE
    │
    ▼
 Execute Block

 FALSE
    │
    ▼
 Skip Block
```

---

# 3. if-else Statement

Executes one block when condition is true and another when false.

Syntax:

```java
if(condition) {

}
else {

}
```

Example:

```java
int age = 16;

if(age >= 18) {

    System.out.println("Adult");
}
else {

    System.out.println("Minor");
}
```

Output:

```text
Minor
```

---

## Execution Flow

```text
Condition
    │
 ┌──┴──┐
 │     │
TRUE FALSE
 │     │
 ▼     ▼
if   else
```

---

# 4. if-else-if Ladder

Used when multiple conditions need to be checked.

Syntax:

```java
if(condition1) {

}
else if(condition2) {

}
else if(condition3) {

}
else {

}
```

---

## Example

```java
int marks = 85;

if(marks >= 90) {

    System.out.println("Grade A");
}
else if(marks >= 80) {

    System.out.println("Grade B");
}
else if(marks >= 70) {

    System.out.println("Grade C");
}
else {

    System.out.println("Fail");
}
```

Output:

```text
Grade B
```

---

## Execution Flow

```text
Condition 1
     │
     ▼
 False
     │
     ▼
Condition 2
     │
     ▼
 True
     │
     ▼
Execute Block
```

Remaining conditions are skipped.

---

# 5. Nested if Statement

An if statement inside another if statement.

Syntax:

```java
if(condition1) {

    if(condition2) {

    }
}
```

---

## Example

```java
int age = 22;
boolean voterId = true;

if(age >= 18) {

    if(voterId) {

        System.out.println("Eligible");
    }
}
```

Output:

```text
Eligible
```

---

## Execution Flow

```text
if(age >= 18)
        │
        ▼
      TRUE
        │
        ▼
if(voterId)
        │
        ▼
      TRUE
        │
        ▼
 Eligible
```

Used inside conditions.

| Operator | Meaning            |
| -------- | ------------------ |
| ==       | Equal To           |
| !=       | Not Equal To       |
| >        | Greater Than       |
| <        | Less Than          |

| <=       | Less Than Equal    |

```java

```

# Logical Operators

Used to combine multiple conditions.

---

## AND (&&)

Both conditions must be true.

```java
if(age >= 18 && citizen == true)
```

Truth Table:

| A   | B   | Result |
| --- | --- | ------ |
| T   | T   | T      |
| T   | F   | F      |
| F   | T   | F      |
| F   | F   | F      |

---

## OR (||)

At least one condition must be true.

```java
if(age >= 18 || permission == true)
```

Truth Table:

| A   | B   | Result |
| --- | --- | ------ |
| T   | T   | T      |
| T   | F   | T      |
| F   | T   | T      |
| F   | F   | F      |

---

## NOT (!)

Reverses the result.

```java
if(!isBlocked)
```

Example:

```java
boolean login = false;

System.out.println(!login);
```

Output:

```text
true
```

---

# 6. Ternary Operator

Short form of if-else.

Syntax:

```java
condition ? value1 : value2;
```

---

## Example

```java
int age = 20;

String result =
(age >= 18) ? "Adult" : "Minor";

System.out.println(result);
```

Output:

```text
Adult
```

---

## Equivalent if-else

```java
if(age >= 18) {

    result = "Adult";
}
else {

    result = "Minor";
}
```

---

## When to Use Ternary Operator?

Good For:

```java
Simple Decisions
```

Avoid For:

```java
Complex Business Logic
```

# 7. switch Statement

Used when one variable needs to be compared against multiple values.

---

## Syntax

```java
switch(variable) {

    case value1:
        break;

    case value2:
        break;

    default:
}
```

---

## Example

```java
int day = 3;

switch(day) {

    case 1:
        System.out.println("Monday");
        break;

    case 2:
        System.out.println("Tuesday");
        break;

    case 3:
        System.out.println("Wednesday");
        break;

    default:
        System.out.println("Invalid");
}
```

Output:

```text
Wednesday
```

---

# Importance of break

Without break:

```java
case 1:
    System.out.println("One");

case 2:
    System.out.println("Two");
```

Output:

```text
One
Two
```

This behavior is called:

```text
Fall Through
```

---


# Modern Switch Expression (Java 14+)

Introduced to remove fall-through problems.

Syntax:

```java
String result = switch(day) {

    case "Saturday", "Sunday"
        -> "Weekend";

    default
        -> "Weekday";
};
```

---

## Example

```java
String day = "Sunday";

String type = switch(day) {

    case "Saturday", "Sunday"
        -> "Weekend";

    default
        -> "Weekday";
};

System.out.println(type);
```

Output:

```text
Weekend
```

---

# Traditional Switch vs Modern Switch

| Feature        | Traditional | Modern |
| -------------- | ----------- | ------ |
| break Required | Yes         | No     |
| Fall Through   | Yes         | No     |
| Returns Value  | Difficult   | Direct |
| Cleaner Syntax | No          | Yes    |

# Common Beginner Mistakes

## Using = Instead of ==

Wrong:

```java
if(age = 18)
```

Correct:

```java
if(age == 18)
```

---

## Missing break in switch

Wrong:

```java
case 1:
    System.out.println("One");
```

May cause unwanted execution.

---

## Writing Complex Nested Conditions

Avoid:

```java
if(...)
{
    if(...)
    {
        if(...)
        {
        }
    }
}
```

Makes code difficult to maintain.

# Summary

- Conditional Statements help programs make decisions.

- Java provides if, if-else, else-if ladder, nested if, ternary operator, switch, and modern switch expressions.
