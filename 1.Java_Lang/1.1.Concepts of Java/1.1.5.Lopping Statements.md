# Loops in Java

---

# Introduction to Loops

## What is a Loop?

A loop is a control flow statement that repeatedly executes a block of code as long as a specified condition remains true.

Loops help automate repetitive tasks and are heavily used in:

- Array Traversal
- String Processing
- Pattern Printing
- Searching Algorithms
- Sorting Algorithms
- Data Structures
- Competitive Programming
- Software Development

---

## Why Loops?

Without loops:

```java
System.out.println("Hello");
System.out.println("Hello");
System.out.println("Hello");
System.out.println("Hello");
System.out.println("Hello");
```

With loops:

```java
for(int i = 1; i <= 5; i++) {
    System.out.println("Hello");
}
```

Benefits:

- Reduces Code Repetition
- Improves Readability
- Saves Time
- Easier Maintenance
- Efficient Data Processing

---

# Components of a Loop

Every loop consists of three major components.

## 1. Initialization

Starting value of loop variable.

```java
int i = 0;
```

---

## 2. Condition

Determines whether loop continues.

```java
i < 5
```

Returns:

```java
true
```

or

```java
false
```

---

## 3. Update

Changes loop variable.

```java
i++
```

or

```java
i--
```

---

# Loop Flow

Example:

```java
for(int i = 1; i <= 3; i++) {
    System.out.println(i);
}
```

Execution:

```text
Initialization
      ↓
Condition Check
      ↓
True
      ↓
Execute Loop Body
      ↓
Update Variable
      ↓
Condition Check Again
      ↓
Repeat
      ↓
Condition False
      ↓
Loop Ends
```

# Types of Loops in Java

Java provides four major loop structures.

```text
Loops
 │
 ├── for Loop
 │
 ├── Enhanced for Loop
 │
 ├── while Loop
 │
 └── do-while Loop
```

---

# 1. for Loop

## Definition

The for loop is an entry-controlled loop.

The condition is checked before execution.

Used when number of iterations is known.

---

## Syntax

```java
for(initialization; condition; update) {

    // code
}
```

---

## Example

```java
for(int i = 1; i <= 5; i++) {

    System.out.println(i);
}
```

Output:

```text
1
2
3
4
5
```

---

## Internal Flow

```text
i = 1
↓
1 <= 5
↓
Print 1
↓
i++
↓
2 <= 5
↓
Print 2
↓
Continue...
```

---

## Reverse Loop

```java
for(int i = 5; i >= 1; i--) {

    System.out.println(i);
}
```

Output:

```text
5
4
3
2
1
```

---

# 2. Enhanced For Loop (For-Each Loop)

## Definition

Introduced in Java 5.

Used to traverse arrays and collections.

No index required.

---

## Syntax

```java
for(dataType variable : collection) {

}
```

---

## Example

```java
int arr[] = {10,20,30,40};

for(int num : arr) {

    System.out.println(num);
}
```

Output:

```text
10
20
30
40
```

---

## Advantages

- Simple
- Readable
- Less Error-Prone

---

## Limitations

Cannot access index.

Wrong:

```java
for(int num : arr) {

}
```

Cannot know whether current element is at index 0,1,2...

---

# 3. while Loop

## Definition

Entry-controlled loop.

Condition checked before execution.

Used when number of iterations is unknown.

---

## Syntax

```java
while(condition) {

}
```

---

## Example

```java
int i = 1;

while(i <= 5) {

    System.out.println(i);

    i++;
}
```

Output:

```text
1
2
3
4
5
```

---

## Flow

```text
Condition
   ↓
True
   ↓
Execute
   ↓
Update
   ↓
Condition
```

---

## Real-Life Use

Password Validation

```java
while(passwordIncorrect) {

}
```

Reading File

```java
while(dataAvailable) {

}
```

---

# 4. do-while Loop

## Definition

Exit-controlled loop.

Condition checked after execution.

Runs at least once.

---

## Syntax

```java
do {

} while(condition);
```

---

## Example

```java
int i = 1;

do {

    System.out.println(i);

    i++;

} while(i <= 5);
```

Output:

```text
1
2
3
4
5
```

---

## Special Property

```java
int i = 10;

do {

    System.out.println(i);

} while(i < 5);
```

Output:

```text
10
```

Executed once despite false condition.

---

# Entry Controlled vs Exit Controlled

| Feature           | Entry Controlled | Exit Controlled |
| ----------------- | ---------------- | --------------- |
| Condition Checked | Before Execution | After Execution |
| Minimum Execution | 0 Times          | 1 Time          |
| Loops             | for, while       | do-while        |

---

# Infinite Loop

A loop whose condition never becomes false.

Example:

```java
while(true) {

    System.out.println("Hello");
}
```

Runs forever.

---

## Another Example

```java
for(;;) {

    System.out.println("Infinite");
}
```

Infinite loop.

# Loop Control Statements

Java provides:

```text
break
continue
return
```

---

# break Statement

Terminates loop immediately.

Example:

```java
for(int i = 1; i <= 10; i++) {

    if(i == 5)
        break;

    System.out.println(i);
}
```

Output:

```text
1
2
3
4
```

---

# continue Statement

Skips current iteration.

Example:

```java
for(int i = 1; i <= 5; i++) {

    if(i == 3)
        continue;

    System.out.println(i);
}
```

Output:

```text
1
2
4
5
```

---

# break vs continue

| break                 | continue                |
| --------------------- | ----------------------- |
| Stops loop completely | Skips current iteration |
| Exits loop            | Goes to next iteration  |
| Terminates execution  | Continues execution     |

---

# Nested Loops

Loop inside another loop.

---

## Example

```java
for(int i = 1; i <= 3; i++) {

    for(int j = 1; j <= 3; j++) {

        System.out.println(i + " " + j);
    }
}
```

Output:

```text
1 1
1 2
1 3
2 1
2 2
2 3
3 1
3 2
3 3
```

---

# Pattern Printing Example

```java
for(int i = 1; i <= 5; i++) {

    for(int j = 1; j <= i; j++) {

        System.out.print("*");
    }

    System.out.println();
}
```

Output:

```text
*
**
***
****
*****
```

# Time Complexity of Loops

## Single Loop

```java
for(int i = 0; i < n; i++)
```

Complexity:

```text
O(n)
```

---

## Nested Loop

```java
for(int i = 0; i < n; i++) {

    for(int j = 0; j < n; j++) {

    }
}
```

Complexity:

```text
O(n²)
```

---

## Triple Nested Loop

```java
for(int i = 0; i < n; i++) {

    for(int j = 0; j < n; j++) {

        for(int k = 0; k < n; k++) {

        }
    }
}
```

Complexity:

```text
O(n³)
```

# Interview Questions

### Difference between while and do-while?

| while                   | do-while                |
| ----------------------- | ----------------------- |
| Condition checked first | Condition checked later |
| May execute 0 times     | Executes at least once  |

---

### Which loop is best when iterations are known?

```java
for loop
```

---

### Which loop is best when iterations are unknown?

```java
while loop
```

---

### Which loop executes at least once?

```java
do-while loop
```
